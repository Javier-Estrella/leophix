== Description ==

This plugin creates the additional sections on the front page in inpersttion themes. A easy plugin to single page and post in add shortcode data for themes by inpersttion Themes.  enjoy the full functionality of the Inpersttion For Theme Software by installing the Inpersttion For Theme.
Inpersttion For Theme is a plugin created to enhance the functionality of the WordPress theme created by Inpersttion For Theme software.

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for Inpersttion For Theme
3. Activate Inpersttion For Theme plugin from your Plugins page.
